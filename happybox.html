Touch activated Message Box

For this project I wanted to experiment with structural woodworking and interactive electronics using a Circuit Playground Express (CPX). I built a small wooden box with the intention of embedding the CPX underneath the surface, making it react to touch via conductive screws mounted on top. I constructed the box quickly using end grain legs for added durability and visual interest. While assembling I did run into a minor mishap, splitting a bit of wood during leg attachment, but it didn’t affect the overall function. After mounting the CPX I spent some time troubleshooting the touch input to get it reliably responding to user interaction. Once the functionality was working as intended I finalized the box by attaching the “message paper” on top, completing the interactive MessageBlox.
